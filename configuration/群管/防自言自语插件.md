# 防自言自语插件

::: tip 提示
防止群内自言自语的插件，可以有效防止群内恶意刷屏。
:::

## 用法
```
#见groupManager.config配置文件
BullshitMsgBlocker:  #防止群友自言自语
  enable: false
  blockList:  #管理的群列表，列表中的群才会被检测
    - 913122269
    - 1050663831
  消息缓存池大小: 5 #缓存最近{n}条消息。建议根据实际需要调整。
  每人最大发言数量: 5 #最近{缓存池}消息中，当单个用户发言达到{n}条时，触发禁言
  禁言时长: 180 #禁言{禁言时长}秒
  嘲讽: "检测到群友自言自语，已禁言180秒。"
```
